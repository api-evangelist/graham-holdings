# Graham Holdings

Graham Holdings is a diversified holding company with operations in education, broadcasting, manufacturing, healthcare, automotive, and hospitality. Subsidiaries include Kaplan, Graham Media Group, Hoover Treated Wood Products, Joyce/Dayton, Group Dekko, Forney, Graham Healthcare Group, The Slate Group, Code3, and Clyde's Restaurant Group.
